# German Windows-PC Keyboard Layout for macOS

This project provides a German keyboard layout for macOS that keeps important special characters at the same physical key positions as on a German Windows PC keyboard.

## What You Get

- Familiar `{ } [ ] | \\` positions for Windows-to-macOS switching.
- A transparent, non-executable XML layout definition.
- A distributable macOS `.bundle` in `root/pc-win-de-keyboard.bundle`.

## Tested Hardware

Tested with **MSI Vigor GK30** (external German PC keyboard):

![MSI Vigor GK30 test keyboard](../images/keyboard-msi-vigor-gk30.jpg)

## Dead-Key Status (Current)

Current behavior in the shipped layout:

- Dead-keys: `^`, acute accent key, grave accent key
- Not a dead-key: `~` (outputs immediately)
